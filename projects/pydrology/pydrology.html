Pydrology

Pydrology is a Python library for requesting and processing hydrology-related data streams for use by scientists and others interested in the water cycle. The aim of the project is to continually collect different sources of data and make them available through an API-type application in a standard format. Currently, USGS streamflow data and NOAA SC-ACIS meteorology data requests are available for use.
